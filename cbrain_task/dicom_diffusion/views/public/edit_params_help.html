Diffusion pipeline options


This tool will process DMRI data.

Options:
  • NLMeans sigma: Sigma parameter to pass to the nlmeans algorithm (default: auto estimation)
  • B0 threshold: Threshold used to define which directions are 'b0' directions (default 0)
  • Fiber response function: Fiber response function to me mutiplied by 10**-4 in the CSD reconstruction (default: 15,4,4)
  • Dicom series tag: Tag used to find the good dicom series (if the dicom file contains multiple series) (default: )
